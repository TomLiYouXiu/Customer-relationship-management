# 安全退出

用户在任意的业务页面,点击"退出"按钮,弹出确认退出的模态窗口;用户在确认退出的模态窗口,点击"确定"按钮,完成安全退出的功能.

​	*安全退出,清空cookie,销毁session

​	*退出完成之后,跳转到首页

# 登录验证

登录验证.

​	用户访问任何业务资源,都需要进行登录验证.	

​	*只有登录成功的用户才能访问业务资源

​	*没有登录成功的用户访问业务资源,跳转到登录页面
